**Stop running back to the farm for five pieces of Wood.**

In vanilla Stardew Valley, if you are short by even a single Stone, Robin (or the *Wizard's*) will refuse to start construction. This mod allows Robin to supply those missing materials herself for a fee.

## How it Works

When you open the Carpenter (or magical) menu, the mod checks your inventory against the building's requirements.

* **If you have all the materials:**
  Everything works exactly like the normal game.

* **If you are missing materials:**

  * Robin calculates the cost of the missing items based on their current market value.
  * The "Buy-out" cost is added to the standard building fee.
  * You pay the **Grand Total** in gold, and construction begins immediately.

## Features

### 1. The Information Board

A board appears next to the building menu showing:

* Which materials you are missing and how many.
* The price for the missing supplies.
* The total construction fee.
* **Red Text Warning:** If the total price is more than you currently have in your funds.

### 2. Smart Pricing

The mod respects your progress. If you have professions (like *Blacksmith*) that make materials more valuable, Robin adjusts her supply fees accordingly. You can also force the mod to use **Base Prices** in the settings.

## Configuration

This mod is fully compatible with the **[Generic Mod Config Menu](https://www.nexusmods.com/stardewvalley/mods/5098)**.

You can easily tweak the rules from the main menu or in-game:

* **Enable Mod** (*true*): Enable or disable this mod.
* **Display Info Board** (*true*): Show the material breakdown board next to the menu. If disabled, the cost breakdown appears in the **Build** button tooltip instead.
* **Use Base Prices** (*false*): If enabled, materials always cost their base value. If disabled, costs adjust based on your professions.

## Installation

1. Install the latest version of **[SMAPI](https://smapi.io/)**.
2. *(Optional but highly recommended)* Install **Generic Mod Config Menu** to easily change your settings in game.
3. Download this mod and unzip it into your *Stardew Valley/Mods* folder.
4. Launch the game using SMAPI.

## Uninstallation

1. Delete the mod's folder from your *Stardew Valley/Mods* directory.

## Known Bug

* A visual glitch may briefly cause the build button to appear in its blocked state (semi-transparent) for a fraction of a second before switching to the valid state.
